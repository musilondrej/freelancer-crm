# Changelog

This changelog documents notable changes to Freelancer CRM.

The format is intentionally simple and grouped by release.

## [0.1.0] - 2026-03-16

Initial MVP release.

### Added

- Customer, contact, lead, and lead source management.
- Project management with customizable status workflows.
- Work logging for hourly and one-time activities.
- Billable, non-billable, done, and invoiced activity lifecycle tracking.
- Reusable activity catalog for work types.
- Recurring services for predictable revenue tracking.
- Financial and operational dashboard widgets.
- User preferences for locale, formatting, and CRM defaults.
- Multi-tenant owner scoping across domain models.
- Pest-based test suite and static analysis tooling.

### Notes

- Tagged as the first public MVP baseline.
- Intended for self-hosted use and further iteration.
